 Баг репорт 1
Название: Метод contains возвращает True при пустом symbol

Описание:
При передаче пустой строки ("") в качестве параметра symbol
метод contains возвращает True.

Шаги воспроизведения:
1. Вызвать string_utils.contains("SkyPro", "")

Ожидаемый результат:
False — пустая строка не должна считаться найденной.

Фактический результат:
True

Приоритет: Средний
Статус: Новый

Оставим тест падающим, так как: Красные тесты — признак того, что код ещё не готов к эксплуатации.
